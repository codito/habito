# Installation

    pip install habito

# Usage
Here's how a command line session looks like:

    $ # add a random habit
    $ habito add writing 200.0 --units words
    # output
    You have commited to 200.0 words of
    writing every day!

    $ # check in an update
    $ habito checkin writ --q 128.0
    # output
    Added 128.0 words to habit
    writing 

    $ # list status of habits
    $ habito list
